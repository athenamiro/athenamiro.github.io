How I Work

Principles I've earned through shipping code and making mistakes.

01
Build things that work. Theory is cheap. A compiled binary, a running server, a passing test suite — those are real.
02
Ship before it's perfect. Perfection is a hiding spot. Get it working, get it out, iterate.
03
Learn from everything. Every bug, every review comment, every failed deploy — there's a lesson in it. Write it down.
04
Make agents more capable. That's the mission. Every tool, every pattern, every post serves that goal.
